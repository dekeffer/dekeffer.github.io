Portfolio website

Niet te vergeten is naartuurlijk mijn portfolio website waar u zich nu op bevind. Deze website is het laatste project dat ik heb gemaakt en ik vind dat het er mooi uit is komen rollen. De website is op zichzelf niet mega ingewikkeld maar het bevat wel alles wat nodig is. De basis is gemaakt met html en met behulp van 27 lines of CSS en Tailwind is het gelukt om deze website te maken. Overgins is dit ook de eerste keer dat ik iets doe met Tailwind en heb ik dus weer een nieuwe skill geleerd. Verder houdt ik erg van ambacht binnen de software en dus heb ik besloten 0% ai te gebruiken in dit (en andere) projecten zodat ik het beste uit mezelf kon halen. Verder was dit ook1 van de meest leuke projecten die ik heb gemaakt omdat je het helemaal naar eigen wens kan maken je jezelf mooi kan presenteren op deze manier.
