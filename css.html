CSS

CSS is een pragtige manier van het mooi maken van je website. Want laten we eerlijk zijn, kale html is gewoon lelijk. Daarom kan ik met mijn website maak skills ook gebruik maken van CSS. Ik ben in staat om door middel van classes, id's en tags specifieke stukjes opmaak voor specifieke stukken op de site toe te wijzen. Zo kan ik backgrounds, text-fonts, afbeeldingen, kleuren en nog veel meer aanpassen met css. alles voor een mooie website.
