GitHub

GitHub is een erg handig tool om te beheersen en daarom heb ik dit geleerd. Ik ben nu in staat om in GitHub te werken in een groep zodat al onze codes samen worden gevoegd. Ook kan ik het gebruiken om mijn eigen codes in op te slaan zodat ik altijd overal mijn betanden kan pullen en verder kan werken aan mijn projecten. Ook kan ik gebruik maken van de scrum functie die in GitHub staat. Verder kan ik het gebruiken om andermans codes te lezen en daar van te kunnen leren. En als kers op de taart kan ik een GitHub pages opzetten zoals bij deze portfolio website. Ik maak ook gebruik van GitHub desktop zodat ik makkelijk mijn codes kan pushen en pullen vanuit en naar GitHub.
