Over mij

Mijn naam is Kevin en ik ben een software developer in ontwikkeling. Net als die perfecte website of perfect werkende machine die in ontwikkeling is en waar dus nog aan gewerkt wordt. Ik ben mezelf aan het ontwikkelen op veel verschillende aspecten. Op Vonk waar ik op school zit leer ik veel over nieuwe talen, maar ook al bekende talen en hoe ik er zo veel mogelijk mee kan. De mogelijkheden zijn dan ook eindeloos binnen de software en dat maakt het zo mooi vind ik. Ik ben zelf erg geintresseerd in het maken van websites en (mobiele)aplicaties. Met deze vormen van software kan ik mensen direct helpen door ze een platform te geven waar ze iets aan hebben. Ik vind het leuk om de voorkant van een website of app te maken en er voor te zorgen dan de klant iets moois te zien krijgt. Daarin tegen is de backend erg belangrijk en vind ik dit dus ook erg intressant. Je zou kunnen stellen dat ik een beetje een all-rounder ben. Ik vind van alles leuk om te doen en kan ook van alles wel iets wat het leuk en gevarieerd houdt. Van het volgen van Scrum tot Scrummaster en van software developer tot koffiehaler. Ik werk graag samen met andere specialisten en kan velen rollen innemen. Hopelijk spreek ik u snel!

Projecten


De RoboMeter was een basis project op school wat ik heb mogen uitbreiden voor een extra uitdaging. Het maken van een robot met een microbit en waardes kunnen uitlezen op een website was erg leuk om te doen!

De website voor Mieke Verschoor bloemisten was mijn eerste echt opdracht. Een website maken met een webwinkel functie zonder dat dit problemen ging veroorzaken. het was een leuk project dat ik met Wix heb gemaakt.

De City Bites Festival website is een schoolproject. Helaas is deze nog niet helemaal af. Daarin tegen is het wel een leuk project om mijzelf te testen op het gebied van HTML CSS en JS.

Deze website is ook 1 van mijn top projecten. Ik heb er veel tijd, HTML, Tailwind en meer ingestopt. Het is erg leuk om te kijken hoe je jezelf het beste neerzet op een website en ik ben erg tevreden met het resultaat.
