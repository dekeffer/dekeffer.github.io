Python

Python is een erg brede taal waar ook ik meerdere dingen mee kan maken. Zo kan ik bijvoorbeeld games maken die draaien in de terminal zoals: hoger of lager, maar ook steen, papier, schaar. Ook kan ik met python bijvoorbeeld een microbit aansturen om iets te doen. Ik heb bijvoorbeeld met behulp van python de RoboMeter gemaakt zodat de microbit bewegingen maakt en een input kan meter en kan teruggeven zodat het kan worden weergegeven op een website.
