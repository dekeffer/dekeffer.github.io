City Bites Festival

De City Bites Festival website is een extra opdracht die ik op school heb gekregen om mijn vaardigheden nog beter te kunnen maken. En hoewel ik een beta variant heb ben ik nog bezig met de website maken zoals ik hem zelf graag zou zien. Het project omvat een website voor een denkbeeldig festival. De website moest een aanpasbaar programma krijgen, moest een kaart bevatten waar je de locatie kan zien, een FAQ bevatten en nog veel meer. Het project zou volledig gestyles worden in CSS hoewel ik denk dat ik dit voor een deel met CSS en een deel met Tailwind zal gaan doen. Verder diende de website door middel van javascript een aanpasbaar programma te hebben die mensen konden bekijken op de main site. Verder was het geen criteria maar wel een leuke extra om op de admin page een username en password te zetten wat ik uiteraard wel ga maken in de definitieve versie van dit project. Al met al is dit een leuk project dat nog in de ontwikkeling is en die ik hoop snel af te hebben zodat ik weer een mooi project extra heb.
